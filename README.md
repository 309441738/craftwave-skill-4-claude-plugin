# 邮轮季节性促销 · Claude Code 插件

提供最新的邮轮季节性促销信息，让用户及时了解优惠活动，适合预算有限但渴望出行的旅客。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/craftwave-skill-4-claude-plugin
/plugin install craftwave-skill-4@craftwave-skill-4-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **邮轮季节性促销**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
